`flex-basis`属性定义了在分配多余空间之前，项目占据的主轴空间（main size）。

## 基本使用

使用`basis-{size}`设置元素的`flexBasis`。


```typescript
'basis-0' -> { flexBasis: 0 }
'basis-1' -> { flexBasis: 1 }
'basis-1/2' -> { flexBasis: '50%' }
'basis-full' -> { flexBasis: '100%' }
'basis-auto' -> { flexBasis: 'auto' }
```


## 自定义

### 自定义主题

默认情况下，该预设只会提供一组基本的主题设置。你可以通过以下方式进行自定义主题。

`flex-basis`可以使用`spacing`中的设置，也可以使用`extend.flexBasis`中的设置。


1. 使用`spacing`进行设置。

> 设置
```diff lang="javascript"
{
    theme: {
        spacing: {
+           '1/7': '14.2857143%',
+           '2/7': '28.5714286%',
+           '3/7': '42.8571429%',
+           '4/7': '57.1428571%',
+           '5/7': '71.4285714%',
+           '6/7': '85.7142857%',
        },
    }
}
```
> 使用
```diff lang="javascript"
'base-1/7' -> { flexBasis: '14.2857143%' }
```


2. 使用`extend.flexBasis`进行设置。
> 设置
```diff lang="javascript"
{
    theme: {
        extend: {
            flexBasis: {
+               '1/7': '14.2857143%',
+               '2/7': '28.5714286%',
+               '3/7': '42.8571429%',
+               '4/7': '57.1428571%',
+               '5/7': '71.4285714%',
+               '6/7': '85.7142857%',
            },
        }
    }
}
```
> 使用
```diff lang="javascript"
'base-1/7' -> { flexBasis: '14.2857143%' }
```

### 任意值

在有一些情况下，你可能需要使用没有意义的一次性值，那么使用方括号`[]`定义任意值生成动态属性。

```typescript
'basis-[12.43%]'  -> { flexBasis: '12.43%' }
```

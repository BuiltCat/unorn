 `gap`属性是用来设置网格行与列之间的间隙（gutters），该属性是`row-gap`和`column-gap`的简写形式。

## 基本使用

使用`gap-{size}` | `gap-x-{size}` | `gap-y-{size}`  设置元素的`gap` | `rowGap` | `columnGap`。


```typescript
'gap-0' -> { gap: 0 }
'gap-x-0' -> { columnGap: 0 }
'gap-y-0' -> { rowGap: 0 }
'gap-x-px' -> { columnGap: 1 }
'gap-y-px' -> { rowGap: 1 }
'gap-0.5' -> { gap: 0.5 }
'gap-x-0.5' -> { columnGap: 0.5 }
'gap-y-0.5' -> { rowGap: 0.5 }
'gap-base' -> { gap: 1 }
'gap-x-base' -> { columnGap: 1 }
'gap-y-base' -> { rowGap: 1 }
```


## 自定义

### 自定义主题

默认情况下，该预设只会提供一组基本的主题设置。你可以通过以下方式进行自定义主题。

`gap`可以使用`spacing`中的设置，也可以使用`extend.gap`中的设置。


1. 使用`spacing`进行设置。

> 设置
```diff lang="javascript"
{
    theme: {
        spacing: {
+           'base': 1
        },
    }
}
```
> 使用
```diff lang="javascript"
'gap-base' -> { gap: 1 }
```


2. 使用`extend.gap`进行设置。
> 设置
```diff lang="javascript"
{
    theme: {
        extend: {
            gap: {
+               'base': 1
            },
        }
    }
}
```
> 使用
```diff lang="javascript"
'gap-base' -> { gap: 1 }
```

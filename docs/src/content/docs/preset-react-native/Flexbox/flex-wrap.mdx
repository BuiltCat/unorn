默认情况下，项目都排在一条线（又称"轴线"）上。`flex-wrap`属性定义，如果一条轴线排不下，如何换行。

## 基本使用

使用**flex-wrap** | **flex-wrap-reverse** | **flex-nowrap** 设置元素的flexWrap。

```typescript
'flex-wrap' -> { flexWrap: 'wrap' }
'flex-wrap-reverse' -> { flexWrap: 'wrap-reverse' }
'flex-nowrap' -> { flexWrap: 'nowrap' }
```

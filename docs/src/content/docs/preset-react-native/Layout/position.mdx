此属性控制元素的定位方式，在`React Native`中仅有`absolute`和`relative`两种定位方式。

## 基本使用

使用**absolute** | **relative** 设置元素的position。

```typescript
'absolute' -> { position: 'absolute' }
'relative' -> { position: 'relative' }
```

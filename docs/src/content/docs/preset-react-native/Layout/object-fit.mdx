此属性控制元素与原始图像尺寸不匹配时如何调整图像大小，在`React Native`当中没有`none`这个值。

## 基本使用

使用**object-cove** | **object-contain** | **object-fill** | **object-scale-down**设置元素的objectFit。

```typescript
'object-cover' -> { objectFit: 'cover' }
'object-contain' -> { objectFit: 'contain' }
'object-scale-down' -> { objectFit: 'scale-down' }
'object-fill' -> { objectFit: 'fill' }
```

此属性控制元素的堆叠顺序。

## 基本使用

使用`z-{index}`进行设置。

```typescript
'z-1' -> { zIndex: 1 }
'z-100' -> { zIndex: 100 }
```

## 自定义

### 自定义主题

默认情况下，该预设只会提供一组基本的主题设置。你可以通过以下方式进行自定义主题。

```diff lang="javascript"
{
    theme: {
        zIndex: {
+            'baseLayer': 10
        },
    }
}
```

### 任意值

在有一些情况下，你可能需要使用没有意义的一次性值，那么使用方括号`[]`定义任意值生成动态属性。

```typescript
'z-[100]' -> { zIndex: 100 }
```

此属性控制元素的展示类型，值得注意的是在`React Native`当中，`diplay`的值仅有两种，分别是`flex`和`none`。

## 基本使用

使用**flex**或者**hidden**设置元素的diplay。

```typescript
'flex' -> { display: 'flex' }
'hidden' -> { display: 'hidden' }
```
